# 工事月間工程表ツール（協電機工）

工事の月間工程表を、ブラウザだけで作って印刷できる無料ツールです。
インストール不要・ネット接続なしでも動きます（ファイルを開くだけ）。

- `index.html` … ツール本体（これだけで完結）
- `manifest.webmanifest` / `icon.svg` … スマホ「ホーム画面に追加」でアプリ化するための部品

## 使い方

1. ブラウザ（Chrome / Edge / Safari）で `index.html` を開く
2. 「サンプルを入れて試す」で動きを確認
3. 工事情報 → 表示期間 → 工種 → 作業バー → 印刷（A3 / A4 横）

## iPhone / スマホ

Safariでこのサイトを開き、共有 → 「ホーム画面に追加」でアプリのように使えます。

これは公開用の配布コピーです。元の開発リポジトリ（非公開）から `index.html` を同期しています。
